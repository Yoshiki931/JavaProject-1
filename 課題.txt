課題2　クラスの自己紹介アプリを作ろう

1.「walking1682」ユーザーの「JavaProject-1」リポジトリをフォークする
　※上記のユーザーが見つからない場合は、
　　既にリポジトリをフォークした友達を見つけて、フォークしたリポジトリの所有者から上記ユーザーを見つけてください。

2.フォークしたリポジトリをローカルにクローンする

3.developブランチに切り替える
$ git checkout -b develop origin/develop

4.developブランチから「feature-学籍番号」ブランチを作成する

5.「JavaProject-1」をeclipseにインポートして開く
　ファイル/インポート/Git/既存の/JavaProject-1/
6.「JavaProject-1」の「HelloWorld」クラスにて、「data」配列の該当する行数を書き換える

7.修正内容をコミットする
　※コミット名は「feature #1  学籍番号」とする
　※コミットの内容は「HelloWorld.java」ファイルのみとする。
git add .
git commit -m "feature #1 4161124"
8.ブランチをプシュする

9.GitHubにて、プッシュしたブランチを「walking1682」の「develop」ブランチにプルリクエストする
　※プルリクエストの内容に、「HelloWorld.java」以外の修正が含まれていた場合、却下します！

10.プルリクエストがマージされた後、developブランチを更新してeclipseにて実行確認する
　※「git add upstream」のし忘れに注意
